# Logo da Aplicação

Coloque o logo da aplicação nesta pasta.

## Especificações

- **Nome do arquivo**: `logo.png`
- **Formato**: PNG com transparência
- **Tamanho recomendado**:
  - Largura: 200-400px
  - Altura: 40-80px (proporção horizontal)
- **Fundo**: Transparente
- **Uso**: Exibido no header, lado esquerdo

## Exemplo de uso

O logo é automaticamente carregado no header da aplicação. Se o arquivo não existir, o título em texto será exibido como fallback.

```
public/logo/logo.png → Exibido no header
```

## Dicas

- Use um logo horizontal (landscape) para melhor aproveitamento do espaço
- Certifique-se de que o logo seja legível em fundo claro e escuro
- Mantenha boa resolução para telas de alta densidade (Retina)
